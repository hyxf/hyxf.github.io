adb logcat命令行使用

  • android

通过adb logical将日志输出到文件.

  1. 清空之前日志

    adb logcat -c

  2. 按时间格式输出日志

    add logical -v time > mylog.txt

  3. 安装应用

    adb install my.apk

  4. 卸载应用

    adb uninstall my.apk

  5. 分析mylog.txt日志文件
